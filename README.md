# 📚 EduApp

EduApp is an education management app to manage students, teachers, attendance, and assignments.

## 🚀 Features
- Add & manage students and teachers
- Attendance tracking system
- Assignment upload & submission
- Student image upload

## 🛠️ Tech Stack
- Android Studio (Java/Kotlin)
- Firebase / SQLite

## 📌 Future Scope
- Login system
- Notifications
- Result management
